# Code Snippets
This is a simple web-server built using Express that adheres to the CRUD properties.
Written in TypeScript and structured as an MVC application.

### Technologies

- TypeScript
- Express.js
- Handlebars (HTML templating)
- MongoDB
- TailwindCSS
- Docker
